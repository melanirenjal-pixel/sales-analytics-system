# Sales Analytics System

## Project Description

This project is a simple Sales Analytics System built using Python.
It reads sales data from a text file, processes the data, and displays the output.

This project is created as part of Day 1 learning to understand:

* File handling in Python
* Folder structure
* Running Python projects using VS Code

---

## Folder Structure

```
sales-analytics-system/
│
├── data/
│   └── sales_data.txt
│
├── output/
│
├── utils/
│
├── main.py
├── README.md
└── requirements.txt
```

---

## How to Run the Project

1. Open the project folder in VS Code
2. Open the terminal in VS Code
3. Make sure you are inside the project folder
4. Run the following command:

```
python main.py
```

---

## Output

The program reads sales data from `data/sales_data.txt` and prints it in the terminal.
